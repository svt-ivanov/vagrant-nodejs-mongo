# Basic Vagrant VM for NodeJS development using Virtual Box, Ubuntu 14.04 LTS and MongoDB
## Installation: ##
 - Install Virtual Box
 - Install Vagrant
 - Clone the repository
 - Start vagrant:
```bash 
$ vagrant up 
```

- Login onto the VM: 

```bash 
$ vagrant ssh 
```
- Start NodeJS server:

```bash 
$ node server.js 
```

- Open the browser on your host machine and type in the address bar:

```bash 
http://0.0.0.0:8000 
```
